# Portfolio — Frontend Only (Vite + React + Tailwind v4)

No backend needed. Contact form uses free FormSubmit.co

## Setup
npm install
npm run dev

## Email setup
1. First form submit ke baad FormSubmit tumhe surajofficemail+portfolio@gmail.com par verification bhejega
2. "Activate Form" click karo — bas ek baar
3. Uske baad har message direct inbox me aayega

To change email: src/App.jsx me const EMAIL = '...' edit karo

## Deploy
- Vercel: `vercel` or import GitHub repo
- Netlify: drag & drop dist folder after `npm run build`
